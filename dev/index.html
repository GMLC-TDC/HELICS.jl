HELICS.jl

HELICS.jl is a Julia wrapper for the HELICS package.

Installation

For now, this package is registered, so you can use the following to install:

Open the package manager REPL (using ])

(v1.1) pkg> add HELICS

To install the latest development version, use the following from within Julia:

(v1.1) pkg> dev HELICS

This package includes HELICS as a library. You do not have to install HELICS separately.

Note that this should work on 32- and 64-bit Windows systems and 64-bit Linux and Mac systems.
